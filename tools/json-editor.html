劇情章節編輯器 ​

以表單與對白卡片編輯 chapterId、多語章節標題，以及每一句的類型／角色欄位／多語台詞，並可上傳／下載同一套 JSON 結構。預設載入倉庫內 story/chapter_day0_prologue.json 的內容。

使用說明

  1. 視覺編輯：調整章節資訊、每一行旁白或角色台詞；可用上移／下移、下方插入、刪除管理順序。
  2. 上傳 JSON：載入既有章節檔後會自動正規化欄位（缺語系則補空字串）。
  3. 下載 JSON：匯出目前編輯結果，檔名依 chapterId。
  4. JSON：需要時可切到 raw JSON，修改後按「套用至表單」同步回視覺編輯。
  5. 重置：還原為倉庫內序章範本（未下載的變更會遺失）。

直接貼上完整章節 JSON 後點「套用至表單」。下載與上傳仍適用同一資料結構。
