序章｜半臉 CG 與番組開場台詞集 ​

本文件整理 戀綜開場蒙太奇 用台詞：可配合半臉 CG、快剪、採訪 VO。細部正式台詞仍以 劇情對話總覽 與 JSON 為準；若與 序章_預告片式分鏡表 鏡頭入場順序不同，以本文件「播映聲部順序」為聲音與字幕編排優先（畫面剪接可另對應別墅入場）。

台詞類型包含：對鏡頭獨白、多句組合、對自己的要求、擇偶條件、參加 show 的動機、可愛 cut-in 等。剪接時可只取其中 1～2 句。

播映聲部順序（女 → 男 → 女） ​

段落順序說明
① 女生第一波搖滾少女 → 清純少女 → 運動少女首發三位女嘉賓先講完（極短標語或多句組前半）。
② 男生體育男 → 暖男 → 高冷男男嘉賓接棒。
③ 女生第二波貴氣大小姐 → 地雷少女（＋番組 cut-in）可選角色；若名額緊，可改為首發三人 第二輪台詞（見 §三）、cut-in 仍以地雷為主。
④ VIP男主角蒙太奇收尾或轉進場前 獨立一聲部。

使用備註 ​

  • 半臉 CG：單句宜控制在約 10～18 字（長句可拆成兩卡）。
  • 節奏：角色之間可用 0.3～0.5 秒 黑場／雜訊切開；地雷少女 可刻意加 0.5～1.2 秒 空白再接下一句，強調頻率錯位。
  • 對照舊分鏡：分鏡表若仍為「三男入場 → 三女入場」，可 畫面跟走位、VO 依本表女→男→女重對；或另改分鏡表與本文件同步。
  • 中間字卡：播半臉 VO 時，畫面正中可疊 大字卡（主標 4～10 字為佳）；下方可再加 副標小字 0.5～1 行。單卡停留約 1.2～2.5 秒，與角色語氣 對拍 或刻意 錯拍（地雷段適合錯拍）。

節目中間字卡（大字標題）建議 ​

與節目名 《禁忌的 Show》 同調；可依 女 → 男 → 女 → VIP 聲部輪替埋卡，或依情緒隨機抽換（同一支預告勿超過 6～9 張 主字卡，避免眼花）。

主標（畫面正中）副標（可選，小字）
打破禁忌的戀綜心動與規則，只能二選一？
越線之前，先心動虛構節目・成年向
禁止在心動時眨眼—
好感，來自禁區這裡沒有安全距離

簽名後，禁止反悔合約／沒收手機，依節目設定

一、女生第一波 ​

1.1 極短一句版（標語／下禎用） ​

角色主句替換句
搖滾少女音量我控；你，閃邊。前半張臉是節目想要的，後半張是我的。
清純少女我、我有準備小點心……如果不介意的話……那個……大家今天都辛苦了……？（愈說愈小聲）
運動少女位子我認了——接下來換你跟上。別客氣，我從不跟猶豫的人握手。

1.2 多句組（要求／擇偶／動機可混剪） ​

搖滾少女 ​

  • 對自己：嘴硬可以，心不能慫。／項圈是我跟自己的約定，不是給誰看的道具。
  • 擇偶：別想用「乖」這個字鎖我；你最好有自己拔不掉的刺。
  • 動機：他們找的是流量；我找的是誰還敢在鏡頭外講真話。
  • 快剪：想追？先別擋我的光。

清純少女（陽光／清純） ​

  • 對自己：今天要笑著進門……（小聲）也、也要把該說的說清楚……
  • 擇偶：我喜歡……會等我講完的人。中途打斷的，我都會記很久。
  • 動機：人家說戀愛綜藝像童話；我想看看童話後面那頁……寫什麼。
  • 碎句：呃、鏡頭在拍對不對？那、那我不可以看錯字嗎……

運動少女 ​

  • 對自己：贏了不囂張，輸了不逃——這樣才配站到場中央。
  • 擇偶：我要能把我逼到喘不過氣還笑得出來的人；弱的不要，假的更不要。
  • 動機：這裡如果全是劇本，我就用成績把它撕開。
  • 狠一點：心動我信；心動還不敢負責的，我直接判出局。

2.1 極短一句版（標語／下禎用） ​

角色主句替換句
體育男鏡頭別晃，我習慣用身體說話。誰想輸，隊形自己排好。
暖男先喝一口吧——熱的，不燙。氣氛繃太緊就不好剪了，對吧？
高冷男好看的照片，通常都只裁一半。社交是選修；生存是必修。

2.2 多句組（要求／擇偶／動機可混剪） ​

體育男 ​

  • 對自己：節奏再快也要呼吸；輸可以，不能輸在沒盡力。
  • 擇偶：我要會跟我槓的人——不是靠嘴，是靠行動把我頂回場上那種。
  • 動機：外面的人都說戀愛靠運氣；我只信汗水的賠率。
  • 壓成一句：我要找的對手，最好也能變成最硬的隊友。

暖男 ​

  • 對自己：茶要泡對溫度，話要說到人不尷尬——這是我給自己的最低標準。
  • 擇偶：我希望她敢把「不舒服」說出口；剩下我來想怎麼把桌子擦乾淨。
  • 動機：有人說溫柔是武器；我只是……不擅長讓場面難看而已。
  • 對話感：先坐吧。今天不管發生什麼，至少別餓著。

高冷男 ​

  • 對自己：我不需要被喜歡；我需要每一步都站得住。
  • 擇偶：漂亮誰都會演；我要的是——演完還敢素顏給我看的人。
  • 動機：這節目把人標好價了；我倒想看，誰會第一件賣。
  • 內斷句：好感可以買；層次不行。

三、女生第二波 ​

3.1 若不做大小姐／地雷：首發三人「第二輪」用法 ​

從 §1.2 每人 另挑一類（例：第一波主打「對自己」，第二波只播「動機」或「擇偶」），或改用 §1.1 替換句 當第二波標語，避免與第一波重句。

3.2 極短一句版（大小姐／地雷，可選） ​

角色主句替換句
貴氣大小姐不用介紹完——我會自己把人看清楚。這種場合嘛……體面留一半就好。
地雷少女……你剛剛說的，和我腦裡的字幕對不上欸。嗯。（停一拍）我還在載入。

3.3 多句組｜貴氣大小姐 ​

  • 對自己：優雅不是慢，是知道什麼時候該停、什麼時候該收。
  • 擇偶：我不需要被崇拜；我需要在亂局裡還把「我選你」說得很乾淨的人。
  • 動機：我家不缺番組邀約；我來，是看誰還值得我親自彎一次腰。

3.4 多句組｜地雷少女（頻率錯位・可愛但難懂） ​

  • 對自己：……反應慢沒關係，錯頻不要硬轉。
  • 擇偶：我要一個，我沉默的時候他也不用填滿空氣的人。
  • 動機：他們說這裡會放大心跳；我想知道，放大之後我還是不是我。
  • 留白接句：你說什麼？（停頓）……喔，那句啊。我記住了。

3.5 地雷少女｜「番組可愛 cut-in」專集 ​

風格：動漫感、可愛、明顯和旁人頻率對不上；可作 一句一卡 或兩句一組（中間空半拍）。

3.5.1 蒙太奇短打 ​

  1. 欸……？剛剛那一段，是廣告還是我的心跳聲？
  2. 你問動機喔？嗯——（久停）我剛剛在想草莓牛奶會不會也叫「粉紅色合約」。
  3. 對擇偶條件嗎？……要、要會陪我把句子講完的那種。啊不是，那句不算，我重講——欸？
  4. 自己對自己的要求……（小聲）每天至少有一次要記得眨眼？
  5. 你說什麼？（認真歪頭）喔，你在講戀愛啊——我以為在講衛星軌道。
  6. 我想來這個 show——因為、因為貓咪夜裡走路沒聲音，但人會；我想聽聽看哪邊比較吵。

3.5.2 番組 cut-in（單句） ​

  1. 欸？這題是加分題嗎？那我要用貓咪語回答可以嗎——喵？
  2. 理想型喔……要會幫我按「存檔」的人！……啊、現實不能存喔？那、那就算……牽手也算存檔？
  3. 動機？我聽說這裡的點心會自己長腳逃跑——我想抓一隻看看。
  4. 對自己的要求：每天至少說一次「今天也很努力」……（小聲）就算只是在床上翻身也算。
  5. 等等、等等——鏡頭轉太快了啦！我的心跳還在上一個 cut！
  6. 擇偶條件……要會陪我等紅燈變愛心的人。（認真）有的地方有，只是大家沒發現。
  7. 我可以撒嬌嗎？……好，我撒完了。下一題。
  8. 為什麼來 show？因為、因為邀請函聞起來像曬過太陽的紙——我喜歡那個味道。
  9. 欸……你剛剛是不是偷偷換題目了？我腦內字幕還停留在「你好漂亮」那格。
  10. 理想約會？去沒人的便利商店，買兩個不一樣的布丁，然後猜誰的比較甜……猜錯的請喝 imaginary 茶。

3.5.3 兩句一組（插播感） ​

  • 大家好～我是來送可愛的！……（停）……啊，送錯棚了嗎？沒關係，可愛先放這。
  • 我對戀愛的信心喔……大概像手機電量 38%。／還能撐，但很想找充電線。
  • 他們說心跳會加速。／我比較像……延遲出貨的那種心跳。

3.5.4 旁人會愣住的接句 ​

  • （正常提問後）好呀好呀——（停）——剛剛那題是選擇題還是天氣預報？
  • 問「為什麼來」→ 因為……欸，門把手的弧度很溫柔？我摸了一下就進來了。
  • 問「理想型」→ 要會讀空氣的——啊，我空氣比較厚，可能要讀久一點；不過沒關係，我可以請你喝 imaginary 茶。

3.5.5 早期版本補錄（可愛 cut-in 系，可併入 3.5.2） ​

  • 欸欸……鏡頭這邊，我可以偷偷比小貓肉球嗎？……不行喔？好，那我只是在腦內比。
  • 欸欸擇偶喔——要會……欸，「會等我載入完」的人？載入、載入中……叮。
  • 動機？我帶了小餅乾。（翻包）……啊，那是給貓的。你也吃嗎？……欸，表情好複雜。
  • 對自己的要求：不要把「早安」講成「晚安」……（想一想）可是有時候天空顛倒也算一種早安吧？
  • 嗯……（超認真）我覺得心動像轉蛋——你抽到 SSR 的時候，旁邊的人還在排隊買礦泉水。

3.5.6 可選「記憶點口癖」（同一蒙太奇挑 1 個重複 2～3 次即可） ​

  • imaginary 茶
  • 腦內字幕
  • 延遲（載入／出貨／心跳）
